6dFGS DR3

When using any of the products based on the 6dFGS DR3 sample, please cite the corresponding data release paper (Jones et al. MNRAS.399 2009, Arxiv:0903.5451).

Wide-angle transformation matrix and window matrix with \Delta k_o = 0.01h/Mpc, \Delta k_{th} = 0.01h/Mpc:
  • M(kth,kth) -> 120 x 200 bins
  • W(ko,kth) -> 200 x 200 bins

    • Wide-angle transformation matrix and window matrix with \Delta k_o = 0.01h/Mpc, \Delta k_{th} = 0.001h/Mpc:
  • M(kth,kth) -> 1200 x 2000 bins
  • W(ko,kth) -> 2000 x 200 bins

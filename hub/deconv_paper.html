Matrial for 'Analyzing galaxy power spectrum measurements' (arXiv:XXXXX)

The products provided here can be handeled using the python modules provided here. Also see the User guide in Appendix A of arxiv:X.

When using any of the products based on the 6dFGS DR3 sample, please cite the corresponding data release paper (Jones et al. MNRAS.399 2009).
Similarly, when using any of the products based on the BOSS DR12 sample, please cite the data release 12 paper (Alam et al. APJ, 219, 12, 2015) and the associated catalog paper (Reid et al. MNRAS, 455 2016).
Finally, when using any of the products based on the eBOSS DR16 sample, please cite the data release 16 paper (Ahumada et al., arxiv:1912.02905) and the associated catalog paper (Ross et al., arxiv:X).

The following products use \Delta k_o = 0.01h/Mpc, \Delta k_{th} = 0.001h/Mpc, meaning the wide-angle transformation matrix M has the shape 1200 x 2000, the window function has the shape 2000 x 200 and the covariance matrix has the shape 200 x 200. The power spectra are measured in smaller bins of \Delta k_o = 0.001h/Mpc and one needs to combine 10 bins so that these power spectra can be used with the other products (see combine_bins parameter in read_power()):

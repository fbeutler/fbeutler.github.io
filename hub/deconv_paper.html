The products provided here can be handeled using the python modules provided here. Also see the User guide in Appendix A of arxiv:X.

When using any of the products based on the 6dFGS DR3 sample, please cite the corresponding data release paper (Jones et al. MNRAS.399 2009).
Similarly, when using any of the products based on the BOSS DR12 sample, please cite the data release 12 paper (Alam et al. APJ, 219, 12, 2015) and the associated catalog paper (Reid et al. MNRAS, 455 2016).
Finally, when using any of the products based on the eBOSS DR16 sample, please cite the data release 16 paper (Ahumada et al., arxiv:1912.02905) and the associated catalog paper (Ross et al., arxiv:X).

The following products use \(\Delta k_o = 0.01h/\)Mpc, \(\Delta k_{th} = 0.001h/\)Mpc, meaning the wide-angle transformation matrix, \(\mathbf{M}\), has the shape 1200 x 2000, the window function matrix, \(\mathbf{W}\), has the shape 2000 x 200 and the covariance matrix, \(\mathbf{C}\), has the shape 200 x 200.
Product Size Description
\(\textbf{M}\) 1200 x 2000 Wide-angle transformation matrix: Transforms from 3 multipoles in the k-range \(0 - 0.4h/\)Mpc in bins of \(\Delta k_{th}= 0.001h/\)Mpc to 5 multipoles with the same binning
\(\textbf{W}\) 2000 x 200 Window function matrix: Transforms from 5 multipoles in the k-range \(0 - 0.4h/\)Mpc in bins of \(\Delta k_{th}= 0.001h/\)Mpc to 5 multipoles with binning \(\Delta k_o= 0.01h/\)Mpc
\(\textbf{C}_{\rm conv}\) 200 x 200 Covariance matrix: For 5 multipoles in the k-range \(0 - 0.4h/\)Mpc in bins of \(\Delta k_{o}= 0.01h/\)Mpc
Mock \(\mathbf{P}^{\rm conv}_{\rm o}\) 2000 Mock power spectrum multipoles: Mock power spectrum multipoles in the k-range \(0 - 0.4h/\)Mpc in bins of \(\Delta k = 0.001h/\)Mpc. You need to combine 10 bins so that these power spectra can be used with the other products (see combine_bins parameter in read_power()).
Data \(\mathbf{P}^{\rm conv}_{\rm o}\) 2000 Data power spectrum multipoles: Data power spectrum multipoles in the k-range \(0 - 0.4h/\)Mpc in bins of \(\Delta k = 0.001h/\)Mpc. You need to combine 10 bins so that these power spectra can be used with the other products (see combine_bins parameter in read_power()).

To be used as $$ \mathcal{L} \propto \exp\left[-\frac{1}{2}(\mathbf{P}^{\rm conv}_{\rm o} - \mathbf{W}\mathbf{M}\mathbf{P}^{\rm true,flat-sky})\mathbf{C}_{\rm conv}^{-1}(\mathbf{P}^{\rm conv}_{\rm o} - \mathbf{W}\mathbf{M}\mathbf{P}^{\rm true,flat-sky})\right] $$ where \(\mathbf{P}^{\rm true,flat-sky}\) is a vector of monopole, quadrupole and hexadecapole models (no wide-angle effects) and \(\mathbf{P}^{\rm conv}_{\rm o}\) is the vector of measured power spectrum monopole, dipole, quadrupole, octopole and hexadecapole. See the user guide in Appendix A of arxiv:X for more details.

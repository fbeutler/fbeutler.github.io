www.benty-fields.com

In 2005 I co-founded the web application www.benty-fields.com, which allows researchers to read and organize arXiv publications in an efficient way as well as to organize organize journal clubs. Currently I am the main developer of this website.

If you want to contribute to benty-fields there are many ways to do so. First, use the feedback option on the bottom left to report any bugs, typos or strange website behavior... usually these problems can be solved very quickly if reported.

If you want to actively help to develop the website get in touch with us at info@benty-fields.com. The website is written in Python and hosted on Github.

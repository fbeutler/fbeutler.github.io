Web application to organize daily publications

In 2015 I co-founded the web application www.benty-fields.com, which allows researchers to read and organize arXiv publications in an efficient way as well as to organize journal clubs. Currently, I am the main developer of this website.

The purpose of this website is to help researchers to organize and find the latest publications most relevant for their research. Benty-fields currently hosts more than 400 journal clubs and has more than 8000 registered users. If you want to contribute to benty-fields there are many ways to do so. First, use the feedback option on the bottom left to report any bugs, typos or strange website behaviour... usually these problems can be solved very quickly if reported.

If you want to actively help to develop the website get in touch with me. The website is written in Python and hosted on GitHub.
